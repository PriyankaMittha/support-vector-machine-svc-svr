# Support Vector Machine Models (SVC & SVR)

This repository contains the implementation of **Support Vector Machine (SVM) models** using **Python and Scikit-learn**.

The project includes both:

* **Support Vector Classifier (SVC)** for classification problems
* **Support Vector Regressor (SVR)** for regression problems

Support Vector Machines are powerful supervised machine learning algorithms used for solving complex classification and regression tasks.

---

## 📌 Project Overview

This project demonstrates:

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Feature scaling
* Training SVM models
* Model prediction
* Model evaluation

---

## 🧠 Algorithms Implemented

### 1️⃣ Support Vector Classifier (SVC)

Support Vector Classifier is used for **classification tasks**.
It finds the **optimal hyperplane** that separates different classes with the **maximum margin**.

---

### 2️⃣ Support Vector Regressor (SVR)

Support Vector Regressor is the **regression version of Support Vector Machines**.

It is used to **predict continuous values** instead of class labels.
